# Resume Bullet Suggestions

Use these bullets on a data engineering, analytics engineering, or platform resume.

---

## Short version (one line)

- Built a Python data contract validator that detects vendor schema drift, enforces YAML-based quality rules, and generates automated impact reports (Pandas, DuckDB, Streamlit, Pytest).

---

## Medium version (detailed data engineering)

- Designed and implemented **AI Data Contract & Schema Drift Guardian**, a modular Python pipeline that validates vendor CSV feeds against YAML data contracts, classifies breaking schema drift, runs rename-aware data quality checks, and exports JSON/Markdown/CSV impact reports for downstream pipeline owners; delivered CLI, Streamlit dashboard, and GitHub Actions CI using **Pandas**, **DuckDB**, **Streamlit**, and **Pytest**.

---

## AI-focused version (rule-based impact summaries)

- Developed a local data observability tool with **rule-based, severity-weighted impact summaries** that translate schema drift and data quality failures into plain-English actions for engineers; combines YAML contract validation, rename detection (`difflib` + abbreviation heuristics), automated reporting, and a **Streamlit** dashboard — **Python**, **Pandas**, **DuckDB**, **Pytest**, no external AI API required.
